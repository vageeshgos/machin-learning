Gyani - Your Personal AI Assistant

Built with Speech Recognition, Weather API, Scheduling, and More

Hi! I'm Gyani. Ask me anything like "what's the weather?"

Overview

Gyani is an intelligent AI voice assistant built in Python. It listens to voice commands, performs speech recognition, speaks responses using text-to-speech, and can interact with APIs and JSON data for schedules and weather.

Technologies Used

Python, SpeechRecognition, SoundDevice, Pyttsx3, Webbrowser, OpenWeatherMap API, JSON, Youtube-DLP

Features

  • Voice command activation with wake word "Gyani"
  • Text-to-speech response engine
  • Weather information via OpenWeatherMap
  • Daily and Weekly AI learning schedules from JSON
  • Search and play songs from YouTube
  • Google and YouTube opening on command
