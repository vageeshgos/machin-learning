Gyani - Your Personal AI Assistant

Speech Recognition, Weather Info, Scheduling, and More

Hi! I'm Gyani. Type commands like "open google" or "what's the weather".

Project Overview

Gyani is a Python-based AI assistant that listens to your voice and performs actions like fetching the weather, opening websites, playing music, and showing study schedules. This web version is a simulation interface that mimics the behavior with text-based interaction.

Features

  • Simulated Wake Word ("Gyani")
  • Voice and text-based commands (via Python speech libraries in local app)
  • Google, YouTube, and song search
  • Weather information via OpenWeatherMap API
  • Dynamic schedule loading via JSON

Explanation

The actual assistant uses Python libraries like SpeechRecognition, Pyttsx3, and SoundDevice to process user voice commands. It uses condition checks and external APIs to perform tasks such as opening browsers, checking weather, and scheduling.

Conclusion

Gyani showcases how voice interfaces and intelligent automation can be combined into one assistant. This project is a practical step towards building real-world AI interfaces like Alexa or Google Assistant using Python.
